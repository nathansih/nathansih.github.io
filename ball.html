Ball and Beam

Mechanical Ball and Beam System

2022

I designed this ball and beam system for a final project in a mechanical design course. An arduino and sensors were used to keep a ball at a certain position on the beam, and this position could be determined by a user input. The ball and beam were supported by an aluminum body.

Tools in a machine shop were used to machine and engrave the aluminum body. Ball and beam components were designed in SolidWorks CAD and then 3D printed. PID code was created and run in Arduino.
